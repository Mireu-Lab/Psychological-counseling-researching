대충 실험 방법 설명 문서

- TEXT 데이터를 BigQuery에 저장 되어있음.
- 그래서 BigQuery에 있는 데이터를 Embeddding 진행 할 거임
- Embedding를 진행하면 안전하게 DataBase에 저장 할 거임
- 저장이 완료 되면 Vector Database에다가 넣어서 Graph 그리다가 대충 데이터의 밀집도를 분석 할 거임
- 분석 완료되는 대로 그걸 Clustering해서 대충 GCN으로 모델 만들어서 구동 할 거임

## 데이터 전처리 프로세싱
|회차|내용|비고|
|:---:|:---:|:---:|
|1차|파일 구조 분류|
|2차|텍스트 문장 분류| (예: "안녕하세요. 반갑습니다", ["안녕하세요.", "반갑습니다"])
|3차|텍스트 임베딩| (사전 학습된 언어 모델을 활용하여 정규화된 텍스트를 벡터로 변환)
|4차|임베딩 데이터 시각화 및 군집 확인| (PCA, t-SNE 등을 사용해 데이터 분포가 연구 가설과 맞게 나뉘는지 눈으로 확인)
|5차|그래프 구조화 및 데이터베이스(pgvector 등) 적재 후 GNN 모델 학습|

모델은 나도 아직 안 그려봤음.
이제 그려 봐야하는데 나도 몰라.
